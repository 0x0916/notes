Go Code: 多返回值

Go语言内置支持函数返回多个值，该特性经常在Go程序中出现，例如，一个函数可以同时返回结果和错误代码。

package main

import "fmt"
 
func numbers() (int, int) {
	return 2, 6
}

func main() {

(int, int) 表明该函数返回两个类型为int的值

 
	a, b := numbers()
	fmt.Println(a)
	fmt.Println(b)

这里我们使用两个不同的值来存储函数返回的值

 
	_, c := numbers()
	fmt.Println(c)
}

如果我们只关心其中的某个返回值，可以用空白定义符_标识不关心的值

 
$ go run multiple-return-values.go 
2
6
6
