Go Code: 指针

Go语言支持指针，允许程序中通过引用传递值或者数据结构。

package main

import "fmt"
 
func zeroval(ival int) {
	ival = 0
}

zeroval使用值传递

 
func zeroptr(iptr *int) {
	*iptr = 0
}

func main() {
	a := 1
	fmt.Println("initial:", a)
	zeroval(a)
	fmt.Println("zeroval:", a)
	b := 1

	fmt.Println("initial:", b)
	zeroptr(&b)
	fmt.Println("zeroptr:", b)
	fmt.Println("ptr:", &b)
}

zeroptr使用指针传递，对一个解引用的指针赋值会改变这个指针引用的真实地址的值

 
$ go run pointers.go 
initial: 1
zeroval: 1
initial: 1
zeroptr: 0
ptr: 0xc42000a2f8

zeroval在main中不能改变a的值，但是zeroptr可以。
